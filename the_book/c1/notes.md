# Cargo
- Det buildtool som används.
    - Check
        - Kompilera (analysera?) utan att bygga
    - Build
        - Bygger till target/debug
        - Icke optimizerat, kan vara långsammare än väntat

        - Kompilerar samt kör programmet (make more or less)
